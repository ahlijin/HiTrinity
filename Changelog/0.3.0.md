# Changelog / 更新日志

## 0.3.0 (2026-06-24)

### 修复

- **pi3 舵机扭矩未使能**：`ros_robot_controller_node.py` 启动时自动执行 `bus_servo_enable_torque(1, 1)`，解决舵机通电不响应位置指令的问题
- **pi3 消息类型不匹配**：`controller_manager.py` 中 `ServoPosition.position` 改为 `float`、`ServoState.position` 保持 `int`，对齐 ROS2 消息定义
- **pi3 struct.pack 参数错误**：`ros_robot_controller_node.py` 中 `set_bus_servo_position()` 将 `i.position` 转为 `int` 再传给 SDK
- **pi3 启动蜂鸣器不响**：`startup_check.py` publish 后增加延时确保 DDS 消息投递
