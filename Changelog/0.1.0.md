# Changelog / 更新日志

## 0.1.0 (2026-06-05)

### 项目状态

HiTrinity — 三位一体智能小车系统，初始规划阶段。整合 HiJetson（语音管线）与 jetson_ws（导航/SLAM）为统一的三机协同架构。

### 已完成

- [x] **三机架构设计文档**
  - 硬件分工矩阵（Jetson GPU / Pi5 SLAM+LLM / Pi3 电机）
  - 完整/降级双模式切换（心跳5s超时自动切）
  - 降级模式内存预算（基线~5.4GB，余量~1GB）
  - ROS2 Topic 全量设计（语音/视觉/融合/跨机）
  - GPU 调度策略（YOLO常驻降帧 + Whisper脉冲抢占）
  - SLAM 地图同步机制（Pi5→Jetson 定期 .pbstream）
- [x] 项目框架搭建
- [x] 远程仓库配置
